🔥 Connect to Firebase

adaptABILITY Scheduler uses Firebase to sync your schedule in real-time across all devices. Setup takes about 5 minutes and is completely free.

Step 1 — Create a Firebase project

Go to console.firebase.google.com → Add project → name it "adaptability-scheduler" → disable Google Analytics → Create.

Step 2 — Add a Web App

In your project → click the </> icon → register the app → copy the firebaseConfig object values into the fields below.

Step 3 — Enable Firestore

In the left sidebar → Build → Firestore Database → Create database → Start in test mode → choose a location near Frankfurt → Enable.
