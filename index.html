🔥 Connect to Firebase

adaptABILITY Scheduler uses Firebase to sync your schedule in real-time across all devices. Setup takes about 5 minutes and is completely free.

Step 1 — Create a Firebase project

Go to console.firebase.google.com → Add project → name it "adaptability-scheduler" → disable Google Analytics → Create.

Step 2 — Add a Web App

In your project → click the </> icon → register the app → copy the firebaseConfig object values into the fields below.

Step 3 — Enable Firestore

In the left sidebar → Build → Firestore Database → Create database → Start in test mode → choose a location near Frankfurt → Enable.

Step 4 — Paste your config below

💬 Slack Integration

Slack notifications post to #schedule via a secure server-side proxy. The webhook URL is stored in Netlify environment variables — nothing to configure here.

What posts to #schedule: session notes (late/early pickup), cancellations, makeup notifications
Managed in-app only: leave requests, approvals, schedule requests

🏖 Leave Management

Run at the end of each calendar year to reset PTO and sick day counters. First-year employees are automatically pro-rated based on their company start date.

Year-End Leave Reset
Click Preview to see what will change before committing.
What this does: Sets PTO Used and Sick Days Used to 0 for all staff. For employees in their first year, Annual PTO is pro-rated from their start date. A snapshot of the old balances is saved to the JSON export before resetting. Export a backup first.
